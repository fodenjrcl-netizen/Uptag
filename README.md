# 📊 Calculadora de Hipótesis y Confidencialidad

Sitio web interactivo desarrollado para un estudiante universitario.  
Permite evaluar hipótesis estadísticas y calcular intervalos de confianza de manera sencilla y visual.

## 🚀 Tecnologías
- HTML5 / CSS3
- JavaScript
- Interactividad con DOM y funciones matemáticas

## 🎯 Funcionalidades
- Ingreso de datos numéricos
- Cálculo de hipótesis estadísticas
- Evaluación de intervalos de confianza
- Interfaz intuitiva y responsiva
- Resultados mostrados en tiempo real
