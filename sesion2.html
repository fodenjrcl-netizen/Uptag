Sesión 2: Prueba de Hipótesis

Conceptos clave

Definición

La prueba de hipótesis es un procedimiento estadístico que permite evaluar si una afirmación sobre una población es razonable, basándose en los datos obtenidos de una muestra. Se formula una hipótesis nula (H₀) que representa una suposición inicial, y una hipótesis alternativa (H₁) que plantea una diferencia o cambio. El objetivo es determinar, mediante cálculos, si los datos apoyan o rechazan la hipótesis nula.

Hipótesis nula y alternativa

Hipótesis nula (H₀): Es la suposición inicial que se considera verdadera hasta que se demuestre lo contrario. Generalmente afirma que no hay efecto, diferencia o relación significativa. Por ejemplo: “El nuevo medicamento no tiene efecto diferente al anterior.”

Hipótesis alternativa (H₁): Es la afirmación que se acepta si los datos muestran evidencia suficiente para rechazar la hipótesis nula. Plantea que sí existe un efecto, diferencia o relación. Por ejemplo: “El nuevo medicamento tiene un efecto diferente al anterior.”

Nivel de significancia (α) y regiones de rechazo

Nivel de significancia (α): Es un valor umbral que indica la probabilidad máxima de cometer un error tipo I, es decir, rechazar la hipótesis nula cuando en realidad es verdadera. Comúnmente se utiliza un valor de α = 0.05, lo que significa que se acepta un 5% de riesgo de equivocarse al rechazar H₀. Este nivel define cuánta evidencia se requiere para considerar que un resultado es estadísticamente significativo.

Regiones de rechazo: Es el conjunto de valores del estadístico de prueba que llevan a rechazar la hipótesis nula. Se determina en función del nivel de significancia y del tipo de prueba (una cola o dos colas). Si el valor calculado del estadístico cae dentro de esta región, se concluye que hay suficiente evidencia para rechazar H₀. En cambio, si el valor cae fuera de la región de rechazo, no se rechaza la hipótesis nula.

Pasos del procedimiento con ejemplos

Paso 1: Plantear la hipótesis

  • H₀: La media de vida útil de una batería es 100 horas.
  • H₁: La media de vida útil de una batería es diferente de 100 horas.

Paso 2: Seleccionar el nivel de significancia (α)

  • Se elige el nivel de riesgo aceptable para cometer un error tipo I.
  • Ejemplo: α = 0.05 (5%)

Paso 3: Calcular el estadístico de prueba

  • Se determina qué fórmula se usará según el tipo de datos (Z, t, chi-cuadrado, etc.).
  • Ejemplo: Si se conoce la desviación estándar poblacional, se usa la fórmula del estadístico Z.

Paso 4: Establecer la región de rechazo

  • Se define el rango de valores que llevarían a rechazar H₀.
  • Ejemplo: Para α = 0.05 en una prueba bilateral, se rechaza H₀ si Z < -1.96 o Z > 1.96.

Paso 5: Calcular el estadístico de prueba

  • Se sustituyen los valores en la fórmula y se obtiene el resultado.
  • Ejemplo: Z = (x̄ - μ) / (σ / √n)

Paso 6: Tomar una decisión

  • Se compara el valor calculado con la región de rechazo.
  • Ejemplo: Si Z = 2.1, se rechaza H₀ porque 2.1 > 1.96.

Paso 7: Interpretar los resultados

  • Se redacta una conclusión clara y contextualizada.
  • “Con un nivel de significancia del 5%, se concluye que la media de vida útil de la batería es significativamente diferente de 100 horas.”
Fórmulas empleadas (para la media y la proporción)

Para la media:

\[ Z = \frac{\bar{x} - \mu}{\frac{\sigma}{\sqrt{n}}} \]

Donde:

  • \(\bar{x}\): media muestral
  • \(\mu\): media poblacional
  • \(\sigma\): desviación estándar poblacional
  • \(n\): tamaño de la muestra
  • Z: estadístico de prueba

Esta fórmula se utiliza para comparar la media de una muestra con la media poblacional y determinar si la diferencia es significativa.

Para la proporción:

\[ Z = \frac{\hat{p} - p}{\sqrt{\frac{p(1 - p)}{n}}} \]

Donde:

  • \(\hat{p}\): proporción muestral
  • \(p\): proporción poblacional
  • \(n\): tamaño de la muestra
  • Z: estadístico de prueba

Esta fórmula se aplica cuando se desea evaluar si la proporción observada en una muestra difiere significativamente de la proporción esperada en la población.

Ejemplo práctico

Una empresa afirma que sus bombillos duran en promedio 1,000 horas. Un investigador toma una muestra de 36 bombillos y obtiene una media muestral de 980 horas. Se sabe que la desviación estándar poblacional es de 60 horas. ¿Se puede afirmar, con un nivel de significancia del 5%, que la duración promedio es diferente a la afirmada por la empresa?

Paso 1: Plantear la hipótesis

  • H₀: μ = 1000 (la media es igual a 1000 horas)
  • H₁: μ ≠ 1000 (la media es diferente de 1000 horas)

Paso 2: Seleccionar el nivel de significancia (α)

  • α = 0.05 (5%)

Paso 3: Calcular el estadístico de prueba

  • Como se conoce la desviación estándar poblacional, se utilizará la prueba Z.
  • Fórmula:
\[ Z = \frac{\bar{x} - \mu}{\frac{\sigma}{\sqrt{n}}} \]

Sustituyendo los valores:

\[ Z = \frac{980 - 1000}{\frac{60}{\sqrt{36}}} = \frac{-20}{10} = -2 \]

Paso 4: Región de rechazo

Para una prueba bilateral con α = 0.05:

  • Se rechaza H₀ si Z < -1.96 o Z > 1.96

Paso 5: Decisión

  • Z calculado = -2
  • Como -2 < -1.96, se rechaza H₀.

Paso 6: Interpretación de resultados

Con un nivel de significancia del 5%, hay evidencia suficiente para afirmar que la duración promedio de los bombillos es diferente a la que declara la empresa.
